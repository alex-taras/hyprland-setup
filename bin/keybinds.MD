# Keybind Reference

## Applications
- `SUPER + RETURN` - Terminal
- `SUPER + SPACE` - App Launcher
- `SUPER + F` - File Manager
- `SUPER + SHIFT + F` - Superfile (TUI File Manager)
- `SUPER + B` - Browser
- `SUPER + SHIFT + B` - Browser (Incognito)

## Webapps
- `SUPER + A` - Claude AI
- `SUPER + SHIFT + W` - WhatsApp Web
- `SUPER + E` - ProtonMail
- `SUPER + M` - Spotify
- `SUPER + V` - Plex

## Window Management
- `SUPER + W` - Close Window
- `SUPER + TAB` - Cycle Next Window
- `SUPER + T` - Fullscreen (Max)
- `SUPER + SHIFT + T` - Toggle Floating
- `SUPER + S` - Toggle Split
- `SUPER + Arrows` - Move Focus
- `SUPER + Mouse Scroll` - Next/Prev Workspace

## Workspaces
- `SUPER + 1-0` - Switch Workspace
- `SUPER + SHIFT + 1-0` - Move to Workspace
- `SUPER + Mouse Drag (Left)` - Move Window
- `SUPER + Mouse Drag (Right)` - Resize Window

## System
- `SUPER + Q` - Toggle Waybar
- `SUPER + SHIFT + Q` - System Menu
- `SUPER + L` - Lock Screen
- `SUPER + K` - Keybind Help (Markdown)
- `SUPER + SHIFT + K` - Keybind Help (Pager)
- `SUPER + O` - Random Wallpaper
- `SUPER + P` - Screenshot (Full Screen)
- `SUPER + SHIFT + P` - Screenshot (Select Area)

## Media Keys
- `Volume Up/Down` - Adjust Volume
- `Mute` - Toggle Mute
- `Brightness Up/Down` - Adjust Screen Brightness
- `Play/Pause` - Media Control
- `Next/Previous` - Track Navigation
- `Power Button` - System Menu
